# vest-frontend (Vue 3 + Vite)

## What’s included
- SPA base with `vue-router` (nested layouts) + `pinia`.
- Folder structure chuẩn để mở rộng:
  - `src/pages/**` (màn hình)
  - `src/layouts/**`
  - `src/components/**`
  - `src/services/**` (axios)
  - `src/stores/**`
- Router đã dựng khung màn hình: Dashboard, Products, ProductDetail, Orders, Customers, Staff, Vouchers, Payments.
- Demo call API: `ProductsList` gọi `GET http://localhost:8080/api/san-pham`.

## Run (local)
```bash
npm i
npm run dev
```
