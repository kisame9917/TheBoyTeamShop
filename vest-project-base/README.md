# Vest project base (BE + FE)

## BE
- `be/vest-backend`
- Spring Boot 3 (Java 17)
- Entities mapping theo file SQL bạn chốt (`ERD_AoVestNam_SQLServer.sql`)
- Layered packages: config / common / exception / domain / repository / service / web

## FE
- `fe/vest-frontend`
- Vue 3 + Vite
- SPA router + layout + screen skeleton

> Lưu ý: Đây là **project base** (khung). CRUD mẫu đang có: `/api/san-pham`.
